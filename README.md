# Projeto Imobiliária Fictícia 🏠

Este é um projeto de demonstração de uma **imobiliária fictícia**, desenvolvido com **Vue 3**, com foco em design moderno e interação de interface. O objetivo é mostrar o layout, scroll horizontal e funcionalidades básicas de navegação. ✨

---

## Tecnologias Utilizadas 🛠️

- **Vue 3** – Framework JavaScript para construção de interfaces reativas.  
- **AOS (`^3.0.0-beta.6`)** – Biblioteca para animações ao rolar a página.  
- **Lucide Vue Next (`^0.577.0`)** – Conjunto de ícones para Vue 3.  
- **CSS** – Estilização personalizada para layout, responsividade e animações.  

---

## Funcionalidades 🚀

- **Menu lateral** para navegação entre seções (`Home`, `Sobre`, `Lotes`, `Contato`).  
- **Scroll horizontal** controlado pelo mouse ou clique no menu.  
- **Seção Lotes** com mapa estático (imagem) e pontos numerados.  
- **Seção Contato** com formulário válido:  
  - Validação de campos (nome, email e mensagem)  
  - Contador de caracteres para mensagem  
  - Ícones de contato com links clicáveis  
- **Responsivo** para desktop e mobile, com ajustes de layout e tamanhos de input.
